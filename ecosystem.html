Deployment & Integration Projects

cloudflare/moltworker

Run OpenClaw (formerly Moltbot, formerly Clawdbot) on Cloudflare Workers. This worker includes a Chrome DevTools Protocol (CDP) shim that enables browser automation capabilities, allowing OpenClaw to control a headless browser for tasks like web scraping, screenshots, and automated testing.

Project History & Evolution

From Clawdbot to OpenClaw

OpenClaw has undergone several rebrandings:

  • Clawdbot - Original project name
  • Moltbot - First rebranding
  • OpenClaw - Current name and branding

Despite the rebrandings and some security challenges along the way, the project has continued to grow and thrive as an open-source initiative.

Ecosystem Scale

The OpenClaw project has achieved impressive scale:

  • Over 145,000 stars on GitHub
  • Over 20,000 forks on GitHub
  • 50+ official integrations with popular services
  • 700+ Skills in the community library
  • Support for 12 major messaging platforms

Key Integrations

OpenClaw supports integrations with numerous services and platforms:

  • Messaging: WhatsApp, Telegram, Discord, Slack, Signal, iMessage
  • AI Providers: Claude, GPT, DeepSeek, and others
  • Smart Home: Spotify, Hue, and other IoT services
  • Productivity: Obsidian, Gmail, GitHub, Twitter
  • Browser Automation: Web scraping, screenshots, automated testing
